# Map local lists/<name>.txt -> mdblist Static List.
#
# Format (one per line, # = comment):
#   <local-name>   <list-id>
#   <local-name>   <username>/<list-slug>
#
# How to find the values:
#   <local-name>  is the prefix of a file in lists/ (without ".txt").
#                 e.g. "netflix-all" matches lists/netflix-all.txt
#   <list-id>     numeric ID — open the list in mdblist and copy the number
#                 from the URL: https://mdblist.com/lists/<user>/<slug>/<id>
#   <user>/<slug> alternative form, both work in the API.
#
# The push_to_mdblist.py script will diff the file against the live list and
# add/remove items so the mdblist list contains exactly the IMDb IDs from
# the local file. Lines below are examples — uncomment and edit.
